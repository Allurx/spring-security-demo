## 三、运行
### 3.1、访问swagger
启动项目，在浏览器访问localhost:9000/swagger-ui.html即可看到swagger页面
### 3.2、用户登录
选择登陆接口，默认的三个用户（admin，user1，user2）的密码都是123456
### 3.3、测试接口
点击swagger右上角的Authorize按钮，将登陆返回的token信息填入，然后对相应的接口进行认证权限测试
